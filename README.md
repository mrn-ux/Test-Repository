# TestProjectAssignment 

Checkout functionality test automation

## Prerequisites

Ensure you have the following installed:

*   [.NET SDK 10.0](dotnet.microsoft.com) latest version
*   [Visual Studio 2026](visualstudio.microsoft.com) with the necessary workloads

## Getting Started

Follow these steps to get the project running locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/mrn-ux/Test-Repository.git
    ```

2.  **Restore dependencies:**
    Open the solution in Visual Studio and build the project, or run the following command in the root directory:
    ```bash
    dotnet restore
    ```

3.  **Run the application:**
    ```bash
    dotnet tests
    ```
  or using Test Explorer in Visual Studio.
